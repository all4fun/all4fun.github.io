使用apt-get 安装NVIDIA官方驱动

发表于2011-06-17|linux
|浏览量:
1. 首先需要添加apt-get源： 
sudo add-apt-repository ppa:ubuntu-x-swat/x-update
  1. 在线安装驱动命令：
    sudo apt-get update
sudo apt-get install nvidia-current nvidia-current-modaliases nvidia-settings
  2. 添加cuda和openCL库的支持（可选）：
sudo apt-get install nvidia-current-dev
